Snake

Игра разработана с использованием фреймворка Qt.

В игре, под управлением игрока, находится змейка, которая перемещается по полю размерностью 10х20 и поедает яблоки, тем самым увеличивая свою длину. Для взаимодействие с игрой доступны следующие клавиши управления:

  • Стрелка влево - изменение направления движения влево, если змейка движется вверх или вниз
  • Стрелка вправо - изменение направления движения вправо, если змейка движется вверх или вниз
  • Стрелка вверх - изменение направления движения вверх, если змейка движется влево или вправо
  • Стрелка вниз - изменение направления движения вниз, если змейка движется влево или вправо
  • P - пауза
  • Q - выход из игры
  • Зажатая клавиша направления движения - ускорение змейки

Установка программы

Установка игры Змейка выполняется с помощью команды make install и производится в каталог src/app

Для запуска змейки в консольном режиме необходимо выполнить команду make snake_cli

Установка игры Тетрис выполняется с помощью команды make install_tetris и производится в каталог src/app

Для запуска тетриса в консольном режиме необходимо выполнить команду make tetris_cli

Удаление игры

Удаление игры выполняется с помощью команды make uninstall.

Архивация

Команда make dist позволяет произвести архивацию всей исходной конфигурации программы.
